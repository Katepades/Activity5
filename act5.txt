GIT Commands
- Git init - initialize folders for repository.
- Git add - insertion of files in Git folders.
- Git add (specific file)-
- Git add . (all modified files)
- Git commit - determine the status of inititalize Git and save all files.
- Git push origin main - upload init files
- Git Remote add origin "repo link.git"
- Git branch -m main 
- Git pull origin - to pull or update the local files.
